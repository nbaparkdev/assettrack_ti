1. Acesso e Autenticação

LOGIN_METHODS

Existem duas formas de acessar o sistema:

Login Tradicional

  1. Acesse a página de login (`/login`).
  2. Digite seu Email e Senha.
  3. Clique em Entrar.

Login via QR Code

Ideal para tablets e acesso rápido.

  1. Clique em "Login com QR Code".
  2. Aponte a câmera para o Crachá Digital.
  3. Digite seu PIN (4-6 dígitos).

3. Gestão de Ativos

ASSET_MANAGEMENT

Consultar Ativos

Acesse o menu "Ativos" para ver a listagem completa.

Busca por Nome Filtro por Status Histórico Completo

Transferência de Responsabilidade

  1. Vá nos detalhes do ativo e clique em "Transferir".
  2. Selecione o Novo Responsável.
  3. O sistema gera uma solicitação pendente de aprovação.
  4. O novo usuário (ou admin) deve aceitar para concluir.

4. Manutenção

WORKFLOW_TI

Passo 1: Solicitação

Usuário reporta problema no ativo via menu "Solicitar Manutenção".

Passo 2: Triagem

Técnico visualiza fila e aceita o chamado.

Passo 3: Reparo

Técnico registra peças e serviços. Ativo fica em status "MANUTENCAO".

Passo 4: Entrega

Técnico conclui e usa o Scanner para devolver o ativo ao usuário.

7. Service Desk

HELP_DESK_TICKETS

Abertura de Chamados

Para solicitações de suporte que não envolvem reparo físico:

  1. Acesse "Service Desk" no menu principal.
  2. Clique em "Novo Chamado".
  3. Selecione o serviço, defina a prioridade e descreva a solicitação.
  4. Opcional: anexe imagens como print de erro (suporte a Ctrl+V para colar).
  5. O sistema gera um código rastreável no formato CH-2026-XXXX.

Acompanhamento e Timeline

  • → Cada chamado possui uma timeline cronológica com todas as interações.
  • → Técnicos e usuários podem enviar mensagens e anexar fotos.
  • → QR Code individual no topo do chamado para acesso rápido via celular.

Dashboard Gerencial

Admins e Gerentes têm acesso a gráficos interativos com análise de status, prioridades, categorias e ranking de usuários. Filtros avançados permitem pesquisar por código, status, categoria e intervalo de datas.

8. Fornecedores

SUPPLIER_MANAGEMENT

Cadastro de Fornecedores

Gerencie os fornecedores vinculados aos ativos do inventário.

  1. Acesse "Fornecedores" no menu (Admin/Gerente).
  2. Clique em "Novo Fornecedor".
  3. Preencha CNPJ, contato, telefone e endereço.

Auto-Preenchimento via NF-e XML

Para agilizar o cadastro, faça upload do arquivo XML da Nota Fiscal:

  1. No formulário de novo fornecedor, clique em "Upload NF-e XML".
  2. Selecione o arquivo XML da nota fiscal.
  3. O sistema extrai automaticamente: razão social, CNPJ, endereço, itens e valores.
  4. Confira os dados e salve.

Histórico de Notas Fiscais

Cada fornecedor mantém um histórico de notas fiscais vinculadas. Acesse os detalhes do fornecedor para visualizar e gerenciar as NF-es associadas.

9. Relatórios de Ativos

ASSET_REPORTS

Relatórios Gerenciais

Acesse "Ativos > Relatórios" para gerar relatórios completos do inventário.

Data Início/Fim Categoria Fornecedor NF-e E-Patrimônio

Exportação PDF

Após aplicar os filtros desejados, clique em "Exportar PDF" para baixar o relatório formatado. O PDF inclui tabela de ativos com E-Patrimônio, categoria, fornecedor e NF-e.

Categorias de Ativos

Admins e Gerentes podem gerenciar categorias em "Ativos > Categorias". Crie categorias como Notebook, Monitor, Switch para organizar o inventário. As categorias ficam disponíveis como filtro nos relatórios e no cadastro de ativos.
